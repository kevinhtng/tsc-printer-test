TSC Label Printer // Protocol Handler Test

NO IE REQUIRED
✓ How this works This page calls tscprint:// — a custom URL protocol registered on this PC. Chrome/Edge hand it off to tscprint.vbs which sends raw TSPL commands directly to the printer over USB. No plugins, no ActiveX, no Internet Explorer.
Setup Checklist — Do These First
01
Create the folder C:\TSCPrint\ on this PC.
02
Copy tscprint.vbs into C:\TSCPrint\.
03
Edit install.reg if you put the .vbs somewhere other than C:\TSCPrint\tscprint.vbs. Open it in Notepad and update the path.
04
Double-click install.reg → click Yes on the UAC prompt → click Yes on the registry confirmation. Done — one time per PC.
05
Open Device Manager → Printers → note your TSC printer's port. Default in the script is USB002. If yours is different, open tscprint.vbs in Notepad and change the printerPort line.
06
Fill in the test data below and click ▶ Print Label. Chrome/Edge will show a popup asking if it's OK to open the protocol — click Open. The script runs silently and the label prints.
